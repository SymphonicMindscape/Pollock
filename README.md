# Pollock
Self-hosted website for scheduling game nights.

### Current Plan
1. Home Screen with a list of upcoming events so everyone on the site can see when the next D&D session (or similar) is.
2. Event Creation Screen with abillity to lay out details and potential dates for the event.
3. Event Details Screen should be generated from a render template for each ongoing event, give details about when, where, and what to bring (think when John elects to cook us dinner)?

#### Future Additions
1. Once an event is created, send an email out to everyone to let them see available days to suggest for the event.
2. Email will direct to the Event Details Screen for the event, where user can then vote on dates for event.
